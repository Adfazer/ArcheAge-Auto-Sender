## ✨ Возможности

### 📦 Для страницы Корзины (`/cart`)
- ✅ **Пакетная отправка** — отправка предметов группами по 5 штук с задержкой между пачками
- 🎯 **Умный выбор** — выбор всех, снятие выбора, инвертирование, выбор до N-го предмета
- 🏷️ **Выбор по названию** — выберите название из списка, и будут отмечены только предметы с этим названием (с остальных выбор снимается). Одинаковые названия с разным количеством (например, «Осколок предела ×30» и «Осколок предела ×50») считаются разными стопками
- ⏳ **Учёт таймера передачи** — предметы с пометкой «Можно передать через: N мин.» подсвечиваются как недоступные, не выбираются и не отправляются; их количество видно в статистике
- 🗑️ **Автоудаление** — успешно отправленные предметы автоматически удаляются из таблицы
- 📊 **Прогресс-бар** — визуальное отображение процесса отправки
- 🛑 **Аварийная остановка** — кнопка остановки в любой момент

### 🔑 Для страницы Пин-кодов (`/pin/activate`)
- 📁 **Загрузка из файла** — поддержка `.txt` файлов со списком пин-кодов
- 🛒 **Импорт из FunPay** — пины автоматически подставляются при переходе со страницы заказа FunPay
- 📜 **Прокручиваемый список** — визуальный список всех пинов с статусами обработки
- ✅ **Настоящая проверка результата** — скрипт разбирает ответ сервера и различает «успешно активирован», «некорректный пин-код», «пин уже активирован» и т.д. (сервер отвечает HTTP 200 даже на невалидные пины, поэтому по коду ответа судить нельзя)
- 🎨 **Цветовая индикация**:
  - 🔵 Синий — в очереди
  - 🔴 Красный — обработка
  - 🟢 Зеленый — успешно активирован
  - 🟠 Оранжевый — ошибка (с причиной: некорректный пин, уже активирован и т.д.)
  - 🟡 Желтый — ответ сервера не распознан, проверьте вручную
- ⏱️ **Задержка между запросами** — 2 секунды между активациями (анти-DDoS)
- 📈 **Статистика** — счетчики обработанных, успешных и ошибочных пинов

### 🛒 Для страницы заказа FunPay (`funpay.com/orders/...`)
- 🔍 **Автопоиск пинов** — скрипт находит все пин-коды в блоке «Оплаченные товары»
- 💾 **Скачивание файла** — кнопка сохраняет `pins_<номер заказа>.txt` для активации в любой момент
- 🚀 **Активация в один клик** — кнопка открывает archeage.ru/pin/activate и автоматически подставляет пины из заказа, останется нажать «Начать активацию»
- 📋 **Копирование** — все пины в буфер обмена одной кнопкой

## 🚀 Установка

### Шаг 1: Установите расширение Tampermonkey
- [Chrome](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
- [Firefox](https://addons.mozilla.org/firefox/addon/tampermonkey/)
- [Edge](https://microsoftedge.microsoft.com/addons/detail/tampermonkey/iikmkjmpaadaobahmlepeloendndfphd)

### Шаг 2: Установите скрипт
1. Откройте **[ссылку на установку](https://raw.githubusercontent.com/Adfazer/ArcheAge-Auto-Sender/main/archeage-universal-tool.user.js)** — Tampermonkey сам откроет окно установки
2. Нажмите **Установить**

> 🔄 **Автообновления:** при установке по ссылке выше Tampermonkey будет сам проверять новые версии скрипта (метаданные `@updateURL`/`@downloadURL`) и предлагать обновление — ничего делать не нужно.
>
> ⚠️ Если вы раньше устанавливали скрипт копированием кода через «Создать новый скрипт» — удалите старый скрипт и установите его один раз по ссылке, иначе автообновления работать не будут.

## 📖 Использование

### 📦 Управление корзиной

<img src="image.png" alt="Компьютер" width="500" height="300">

*Рис. 1: Панель управления корзиной в правом верхнем углу экрана*

1. **Перейдите** на страницу [archeage.ru/cart](https://archeage.ru/cart)
2. **Дождитесь появления** панели управления (правый верхний угол)
3. **Выберите персонажа** из выпадающего списка
4. **Выберите предметы** одним из способов:
   - Используйте кнопки "Выбрать все" / "Снять выбор"
   - Введите число N в поле "До №" для выбора первых N предметов
5. **Нажмите** 🚀 **"Отправить выбранные"**

<img src="image2.png" alt="Компьютер" width="500" height="300">

*Рис. 2: Процесс отправки пачек предметов с индикатором прогресса*

&gt; ⚠️ **Важно:** После успешной отправки предметы автоматически исчезают из таблицы сайта. Ошибочные пачки остаются выделенными.

### 🔑 Активация пин-кодов

<img src="image3.png" alt="Компьютер" width="500" height="300">

*Рис. 3: Интерфейс активации пин-кодов с загруженным списком*

1. **Подготовьте файл** `pins.txt` со списком пин-кодов (по одному на строку):
2. **Перейдите** на страницу [archeage.ru/pin/activate](https://archeage.ru/pin/activate/)
3. **Нажмите** 📂 **"Выбрать файл .txt"** и загрузите ваш файл(Каждый пин в файле с новой строки)
4. **Нажмите** ▶️ **"Начать активацию"**

> 💡 **Совет:** Список автоматически прокручивается к текущему обрабатываемому пину. Вы можете остановить процесс кнопкой ⏹ в любой момент.

> ⚠️ **Важно:** перед активацией авторизуйтесь на archeage.ru — иначе все пины получат статус «Требуется авторизация на сайте».

---

### 🛒 Активация пинов из заказа FunPay

1. **Откройте страницу заказа**, например `https://funpay.com/orders/XXXXXXXX/`
2. **Дождитесь появления** панели «🛒 FunPay → ArcheAge» (правый верхний угол) — скрипт сам найдет пин-коды в блоке «Оплаченные товары»
3. Дальше два варианта:
   - 🚀 **"Активировать на archeage.ru"** — откроется страница активации, пины подставятся автоматически, останется нажать ▶️ **"Начать активацию"**
   - 💾 **"Скачать pins.txt"** — сохранит файл с пинами для активации позже (через обычную загрузку файла)

> 💡 Переданные пины «живут» 10 минут — если страница активации не была открыта за это время, просто нажмите кнопку в заказе еще раз.
